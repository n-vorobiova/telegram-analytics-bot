# Асинхронний Telegram-бот для бізнес-аналітики та обробки даних

Комерційно орієнтований Telegram-бот на Python, розроблений для автоматизації збору, валідації та статистичного аналізу числових масивів даних (фінансові показники, складські залишки, щоденні бізнес-метрики).

## 🚀 Функціонал
* **Smart Parsing:** Автоматичне розпізнавання та очищення числових масивів із будь-якого скопійованого тексту.
* **Багаторівнева валідація:** Захист від некоректного введення даних, текстового спаму та зависань.
* **Бізнес-аналітика:** Розрахунок ключових метрик (Сума, Середнє значення, Медіана, Мінімум/Максимум) за 1 секунду.
* **Візуалізація звітів:** Автоматична генерація та надсилання графіків динаміки показників (використовуючи Matplotlib/Numpy) без збереження зайвих файлів на сервері.
* **Зручний UX/UI:** Інтуїтивне керування через інтерактивні клавіатури месенджера.

## 🛠 Технологічний стек
* **Мова:** Python 3 (Asyncio)
* **API:** aiogram 3.x
* **Аналіз та візуалізація даних:** Matplotlib, NumPy, Pandas
* **Контроль версій:** Git
